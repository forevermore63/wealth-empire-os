## Quick Start (Recommended)
1. Open this repo in GitHub Codespaces (green "Code" button → Codespaces → Create new).
2. In the terminal inside Codespaces run:
   ```bash
   npm install
   shopify login --token YOUR_ATKN_TOKEN
   shopify app config link
   shopify app dev
   ```
3. Connect your dev store and start building.
